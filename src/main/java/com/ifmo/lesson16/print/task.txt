Реализуйте команды для сервера с помощью интерфейса Externalizable.
1. /ping - По этой команде клиент шлет и принимает ping сообщение 5 раз подряд.
  Высчитывает среднее вермя отклика и печатает в консоль.
2. /server_time - По этой команде сервер возвращает сообщение со своим текущим временем.
3. /list_users - Сервер возвращает список имён всех пользователей, которые подключались за время работы сервера.
4. /ban ip - После ввода этой команды сервер больше не печатает сообщения и не выполняет команды с этого адреса.
   /unban ip - Отменяет действие предыдущей команды.
